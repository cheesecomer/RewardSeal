# ごほうびシール

幼児向けの「頑張ったねシール」をデジタル化した Android アプリです。

子どもが毎日の習慣やお手伝いを達成すると、シール（スタンプ）を集めることができます。
ゴールまで到達すると、事前に設定したごほうびを獲得できます。

## 概要

親が以下の内容を設定してシートを作成します。

- 何を頑張るか
- 何回頑張るか
- ごほうびは何か

作成されたシートは双六のようにマスで構成され、達成するたびにスタンプを押して進めます。

## 対象

- 幼児
- 小学校低学年
- 子育て中の保護者

## 主な機能

### シート作成

- タイトル入力
- 目標回数設定
- ごほうび設定
- スタンプ種類選択

### シート管理

- 複数シート作成
- シート一覧表示
- シート編集
- シート削除

### スタンプ

- 1回達成ごとにスタンプを押す
- 達成状況の可視化
- 双六形式で進捗表示

### ごほうび

- ゴール到達時の演出
- ごほうび表示
